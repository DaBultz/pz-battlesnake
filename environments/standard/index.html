Standard/Duel#

Environment Creation#

Standard Game of 4 Players:

from pz_battlesnake.env import standard_v0

env = standard_v0.env()

env.reset()
for agent in env.agent_iter():
    observation, reward, done, info = env.last()
    action = policy(observation, agent)
    env.step(action)

Duel Game of 2 Players:

from pz_battlesnake.env import standard_v0

env = standard_v0.env(num_agents=2)

env.reset()
for agent in env.agent_iter():
    observation, reward, done, info = env.last()
    action = env.action_space(agent).sample()
    env.step(action)

Observation Space#

The observation space matches the API request from battlesnake, you can find this in their official API Docs.

Action Space#

The Action space consists of a Move, which has 4 diffrent strings as options:

  • up

  • down

  • left

  • right
